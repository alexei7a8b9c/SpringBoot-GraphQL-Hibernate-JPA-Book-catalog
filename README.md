Book Manager Application is a comprehensive full-stack web solution built with Spring Boot 3.2.0 as the backend framework and pure HTML/CSS/JavaScript for the frontend, featuring a modern GraphQL API that connects to a PostgreSQL database through Spring Data JPA with Hibernate for object-relational mapping. The backend implements a complete CRUD system with RESTful principles and GraphQL query/mutation endpoints, while the frontend offers a responsive, intuitive interface with real-time search, pagination, and form validation, all styled with CSS3 Grid/Flexbox and Font Awesome icons, deployed as a single Maven-managed application
